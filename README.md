## Aerospace Public Sentiment Analyzer
This project uses Python and Natural Language Processing (NLP) to analyze text data related to SpaceX and Starlink. It utilizes TextBlob to perform sentiment analysis, automatically categorizing public text as positive, negative, or neutral.

**Technologies Used:**
* Python
* Pandas (Data structured processing)
* TextBlob (Natural Language Processing & Sentiment Analysis)
* Jupyter Notebooks
